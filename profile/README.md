# ISB Dispatch

A student-led media project from the International School of Beijing, delivering news and updates through both a website and an iOS app. Active for over a year, ISB Dispatch continues to grow and evolve.

---

## Overview
ISB Dispatch delivers news and updates via a modern, responsive, and user-friendly platform. Accessible on both web and iOS, it ensures readers can stay connected anytime, anywhere. This project reflects over a year of continuous development, iteration, and learning.

---

## Purpose
This project provides hands-on experience in web and app development, including interface design, dynamic content management, and real-world functionality while engaging a live audience.

---

## Future Plans
- Maintain and optimize both website and iOS app  
- Introduce a finance system and coupon features  
- Expand to Android and MR platforms  
- Enhance user experience and performance  
- Add advanced content management and interactive features
